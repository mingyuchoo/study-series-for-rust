# window-app

Create GPUI App으로 생성된 윈도우 기반 gpui 애플리케이션입니다.

## 주요 의존성

- **gpui 0.2.2**: GPU 가속 UI 프레임워크
- **gpui-component 0.5.1**: gpui 컴포넌트 라이브러리
- **rust-embed 8.11.0**: 정적 자산 임베딩
- **anyhow 1.0.102**: 에러 처리

## 실행 방법

- Rust가 설치되어 있어야 합니다 - [Rustup](https://rustup.rs/)
- `cargo run` 명령으로 실행
